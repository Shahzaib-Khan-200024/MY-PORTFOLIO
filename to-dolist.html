To-Do List Application

This To-Do List application is a user-friendly tool designed to help users organize their tasks efficiently. Built using HTML, CSS, and JavaScript, the app allows users to create, update, and delete tasks in real time. Key features include task prioritization, a clean and responsive interface, and an intuitive experience on both desktop and mobile devices. JavaScript was used to implement dynamic functionalities, such as marking tasks as completed and saving data locally, ensuring user tasks persist even after page reloads. This project demonstrates my ability to create interactive and responsive web applications with attention to functionality and user experience.
